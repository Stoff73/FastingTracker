## Project Overview

FastingTracker is an iOS fasting tracker app built with SwiftUI and SwiftData. Features include: circular progress timer with fasting stages, mood tracking, friends/social fasting, learning center, notifications, Face ID profile lock, and social media sharing. Data is stored locally via SwiftData.

## Build & Test Commands

```bash
# Build (use iPhone 17 Pro — no iPhone 16 in this simulator set)
xcodebuild -scheme FastingTracker -destination 'platform=iOS Simulator,name=iPhone 17 Pro' -configuration Debug build

# Run tests
xcodebuild -scheme FastingTracker -destination 'platform=iOS Simulator,name=iPhone 17 Pro' test

# Run UI tests only
xcodebuild -scheme FastingTracker -destination 'platform=iOS Simulator,name=iPhone 17 Pro' -only-testing:FastingTrackerUITests test
```

Open `FastingTracker.xcodeproj` in Xcode for full IDE support (previews, simulator, debugging).

## Architecture

- **UI Framework:** SwiftUI with tab-based navigation (5 tabs: Home, Friends, Notifications, Learn, Profile)
- **Data Layer:** SwiftData with three models: `FastingSession`, `UserProfile`, `Friend`
- **State Management:** `@Observable` `FastingManager` drives the timer and fasting logic; injected into views
- **Authentication:** `AuthenticationManager` wraps LocalAuthentication (Face ID / Touch ID / passcode)
- **Notifications:** `NotificationManager` singleton schedules milestone and check-in notifications via UNUserNotificationCenter
- **Sharing:** `SocialShareManager` generates share text for start/during/end moments; `ShareSheet` wraps UIActivityViewController
- **Testing:** Swift Testing framework (`import Testing`, `@Test` macro), not XCTest
- **Info.plist:** Auto-generated by Xcode. Privacy keys (NSFaceIDUsageDescription, NSContactsUsageDescription) set via INFOPLIST_KEY_ build settings in pbxproj.

## Project Structure

```
FastingTracker/
  FastingTrackerApp.swift          # Entry point, ModelContainer setup, notification init
  Models/
    FastingSession.swift           # @Model: fasting session with mood log, stages, progress
    FastingStage.swift             # Fasting stages data (Digestion → Deep Cleanse) with colors/descriptions
    UserProfile.swift              # @Model: user stats, goals, streaks
    Friend.swift                   # @Model: friend with fasting status, authorization
    LearnArticle.swift             # Static learning content articles
  Services/
    FastingManager.swift           # @Observable: timer, start/end fast, CRUD, profile stats
    NotificationManager.swift      # Singleton: schedule/cancel fasting milestone notifications
    AuthenticationManager.swift    # @Observable: Face ID/Touch ID/passcode auth
    SocialShareManager.swift       # Share text generation + UIActivityViewController wrapper
  Views/
    MainTabView.swift              # Root TabView with 5 tabs
    Home/HomeView.swift            # Main fasting screen: progress ring, timer, mood, controls
    Friends/FriendsView.swift      # Friends list, add friend, fasting-together status
    Notifications/NotificationsView.swift  # Notification feed
    Learn/LearnView.swift          # Article list with categories, fasting stages reference
    Profile/ProfileView.swift      # Face ID lock, stats, history, settings
    Components/
      CircularProgressView.swift   # Animated progress ring with stage markers
      MoodSelectorView.swift       # Emoji mood picker
```
